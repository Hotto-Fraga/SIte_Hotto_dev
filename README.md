# Hotto Dev

Portfolio website showcasing development skills and interactive demos.

## What's Inside

**Tech Stack Display**: 9 technologies I work with (Kotlin, Java, Python, C++, Tailwind, React, JS, Next, Node)

**Mini Projects**: 6 working demos you can try
- Weather App - shows weather conditions  
- Calculator - basic math operations
- Todo List - task manager
- Timer - stopwatch functionality
- Color Picker - grab colors with HEX/RGB values
- Quote Generator - random programming quotes

**Tech Pages**: Click any technology to read more about it

## Tech Used

- Next.js 16 (App Router)
- React 19
- TypeScript 5
- Tailwind CSS 4
- next-themes (dark mode)
- react-icons

## Running Locally

```bash
npm install
npm run dev
```

Site runs on `http://localhost:3000`

## Project Structure

```
src/
├── app/              # Pages and routes
│   ├── page.tsx      # Homepage
│   ├── layout.tsx    # Root layout
│   └── tech/         # Technology detail pages
├── components/       # React components
│   ├── HottoDev/     # Hero and tech stack
│   ├── MiniProjects/ # Interactive demos
│   ├── NavBar/       # Navigation
│   ├── Footer/       # Footer
│   └── ui/           # UI elements
└── styles/           # CSS files
```
